AUTOMATION PROJECT (Python + Selenium)

What this is
- A Python script that opens a browser and auto-fills a demo form on:
  https://demoqa.com/automation-practice-form

Requirements
- Python 3
- Google Chrome installed
- ChromeDriver installed (must match your Chrome version)

How to run
1) Open terminal
2) cd /Users/bunny/Downloads/pm/automation
3) python3 -m venv venv
4) source venv/bin/activate
5) pip install -r requirements.txt
6) python form_automation.py

Expected output
- It should open Chrome, fill the form, submit, and print:
  Form submitted successfully!
